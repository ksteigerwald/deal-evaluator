This is a web-scraping application for compiling and scoring private equity deals across listing sites based on given preferences, parameters, and keywords. Preferences for these searches have already been programmed into the system. These preferences include types of companies, minimum cash flow, and industry sector.


• To temporarily change these search preferences, visit Websites → Parameters → whichever site you'd like to change preferences for, then make any changes and click "Update Preferences."

• To search by individual site, visit the Homepage (select Evaluator → Home Page, or click "Return to Start" at the bottom of any page) and select the radio button for your choice of site under the header "Choose which site to search from". Then, click "Go" and the program will bring you to the relevant page. From there, you can select which state you want to search in, and click "Submit" to begin the scraping process. Results will appear in the table below, sorted by score.

• To temporarily change the keywords used in the scoring algorithm, visit Scoring Algorithm → Edit, make changes and click "Update preferences." To view the original keyword list, visit Scoring Algorithm → Keywords.

• To search across all sites, visit the Homepage and enter your preferences for cut-off score in the spin box above the page divider. The search results will be filtered so only new (or all, if you select "all" instead in the dropdown box above the spin box) results with a score above that cut-off will be shown in the table.

• To save results to Starred Results, select the results you'd like from the listing table and click the action button "Save to Starred Results." These results will be available later (even after closing the application) from Results → Starred Results.

• To export results to a spreadsheet, select the results you'd like to export from the listings table and click the action buttom "Export" or select Results → Export.

• To view results from previous cross-site scrapes, visit Results → Previous Scrapes.


If you have feedback or suggested improvements, visit Help → Send Feedback and submit your comments or concerns. They will be automatically sent to the developer.
